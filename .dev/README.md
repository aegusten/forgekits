# .dev — Admin Development Space

This folder is the internal workspace for ForgeKits development. It's where ideas live before they become features, where backlogs are tracked, and where experimental tests happen.

**This folder is gitignored from the public repo.**

## Structure

```
.dev/
├── backlogs/        # prioritized feature backlogs
├── ideas/           # raw ideas, brain dumps, exploration
└── tests/           # experimental tests, spikes, prototypes
```

## How to Use
- Drop ideas in `ideas/` as markdown files — messy is fine
- When an idea is validated, move it to `backlogs/` with priority and scope
- Use `tests/` for throwaway experiments before committing to the main codebase
